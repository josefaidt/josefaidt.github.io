ART 4567 - Final Project
December 5th, 2016

Today is the due date for the final critique.


I signed up for this class as an elective to learn the web design portion of front-end web production. Previously I had learned the basic of web development technologies and SEO implementation in another class, and was looking forward to learning web design. In the end I realized I had learned core design concepts while simultaneously obtaining the ability to implement advanced web development languages and tools.

The class began as an intro to web development by running through the essentials of web development, including HTML5 and CSS3. Throughout the course we were assigned to build and design three websites as projects using the following topics: a restaurant, typography, and personal portfolio.

For the restaurant website I chose to develop using HTML and CSS, designing a red and white themed website for my favorite fried chicken restaurant. The typography site was dedicated towards using transition effects and an icon font. This project I began using SCSS for styling in favor of the nesting and mixins opportunities it presents. Finally, the personal portfolio, which is now my website. Before, I used WordPress to build a simple web portfolio. This project I moved on to using Grunt as a task runner primarily for compiling my SCSS files and using Flexbox for the layout of elements.

Overall my experiences with the web development technologies noted above contributed to my existing skill set, thus improving my outlook on websites' designs and how they assist in streamlining transactions. I plan to continue building onto this website to further add to my web production understandings.
