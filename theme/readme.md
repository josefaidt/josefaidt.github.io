# @josefaidt/gatsby-theme

A responsive theme for personal sites and blogs.

## Installation

1. Install the theme

   ```sh
   yarn add @josefaidt/gatsby-theme
   ```

2. Add the theme to your `gatsby-config.js` plugins array:

   ```js
   module.exports = {
     plugins: ['@josefaidt/gatsby-theme'],
   }
   ```
