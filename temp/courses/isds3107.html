Beginning Programming in C# Course Description:  Fundamentals of programming, program design, application development interfaces, debugging, testing, and implementation. Assignments:
  • Two Class Projects
  • Sample Use Case Diagram
  • Program 2 Design Documentation
Course Reflection:  Coming into this class with previous programming knowledge in Python, C++, Javascript, and  HTML/CSS, I was excited to dive into the world of windows form applications. Quickly, the skills I had learned were put to the test as I scoured to understand the syntax of the language. During class time the instructor provided us with practical applications thus solidifying the key points and concepts of the text. This class has ultimately driven me to seek more opportunities with programming as it is becoming a valuable asset in today's technological market.
